## prefix

[initial] : repository를 생성하고 최초의 파일을 업로드 할 때<br>
[add] : 신규파일 추가<br>
[update] : code를 변경할 때<br>
[refactor] : code를 refactoring했을 때(버그 수정이나 기능 추가가 아님)<br>
[fix] : 잘못된 link정보 수정, 필요한 모듈 추가 및 삭제<br>
[remove] : 파일 제거<br>
[style] : 디자인 관련 변경사항<br>
[docs] : 문서 관련 변경(README.md수정)<br>
